This project showcases an interactive Airline Performance Dashboard built using Power BI. It analyzes key airline metrics such as revenue, delays, passenger load, and route performance to generate actionable insights for better decision-making.

📊 Dashboard Features
🔹 Key Performance Indicators (KPIs)

Total Revenue: 12M

Average Delay: 90 minutes

Average Passengers: 191

Most Profitable Route: SFO–TYO

🔹 Visual Insights

📈 Revenue Trend Analysis – Track daily revenue patterns

🛫 Revenue by Route – Compare route-wise performance

🌍 Geographical Map – Visualize revenue distribution

⏱️ Delay Analysis – Compare Domestic vs International delays

🔹 Interactivity

Region filter (Domestic / International)

Date range slicer for dynamic exploration

🛠️ Tools & Technologies

Power BI – Dashboard creation & visualization

DAX – Measures & KPI calculations

Data Modeling – Data transformation & relationships

🎯 Objective

The goal of this project is to demonstrate data storytelling and business intelligence skills by transforming raw airline data into meaningful, interactive insights.
